
### Blockchain Explorer App
Implements the following:
1. Queries blockchain for crypto data
1. Creates a crypto wallet and transacts
1. PHPUnit tests
1. JWT authentication
1. Laravel JSON API (LJA)
1. MYSQL DB
1. Route versioning
